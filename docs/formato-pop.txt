O formato 'pop' foi pensado para entrar gabaritos de acordes de
maneira simples e rápida. Ele tem limitações, mas deve ser suficiente
para corais de Bach.

A sintaxe básica é <acorde><tipo do acorde><acrescimos>/<inversão>,
por exemplo:

G
C/E
C7/E
Cm7/Eb
C!7

Os tipos de acordes são (nada) para maior, m para menor, ° para
diminuto, ø para meio-diminuto, e ! para incompleto (i.e. sem a
terça).

* notas melódicas

A forma [<notas>] expande em (m! <notas>), por exemplo:

[f#] => (m! fis)

* acorde de sétima e extensões

C7m é do com setima maior e C7 é do com setima menor

A forma <acorde>--7 significa que são 2 segmentos onde o segundo pode
ser visto como um acode de sétima do primeiro, mas a sétima também
pode ser uma nota melódica. Essa forma expande para 
(* 2 (<acorde> <acorde com sétima)), por exemplo:

C--7 => (* 2 ((C MAJ 0) (C MAJ 0 7)))

A forma <acorde>_-7 funciona como <acorde>--7, mas quando a sétima
está no baixo:

C_-7 => (C MAJ 0) ((C MAJ 3 7) (m! bes))

A foma <acorde>==7 expande em dois acordes onde o segundo tem a
sétima:

C==7 => (C MAJ 0) (C MAJ 0 7)

A forma <acorde>__7 funciona como <acorde>==7 mas no segundo segmento
a sétima está no baixo:

C__7 => (C MAJ 0) (C MAJ 3 7)

Outras extensões podem ser entradas separadas por ponto. [esse recurso
ainda não foi completamente implementado. outras extensões além da
sétima serão silenciosamente ignoradas] Por exemplo:

Cm7.9b.13/Eb

* diferentes possibilidades

Para entrar mais de uma possibilidade de análise para um segmento
mínimo, é só colocar as possibilidades entre parênteses:

([f#] Bm)

* Sextas aumentadas

As sextas aumentadas devem ter o sufixo -it, -al, ou -fr depois da
"fundamental" do acorde:

eb-it

* Repetição

A forma <acorde>*2 expande para (* 2 <acorde>), exemplo:

C7*2 => (* 2 (C MAJ 0 7))
